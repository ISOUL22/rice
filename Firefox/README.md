# How to install:

- in `about:config`
  - set `natsumi.tabs.type` and `natsumi.theme.type` to `soulmate`
  - set `natsumi.theme.accent-color` to `mauve`
- make a backup of your local `chrome` folder, then copy the `.css` files from the `Firefox` folder to your `chrome` folder.

*meant for firefox, never tested on floorp or other forks.*
